Watchdog port


The port to run the Watchdog service on.
This is very important, since this system is responsible for two very important things:
- Keeping track of Redundant/Obsolete/Rubbish files
- Generating media previews

Changing this later requires the entire server to be shut down first.

Upload Service Port


Port to run the upload system on.
This system is very important too, since it's responsible for file uploading.

Changing this later requires the entire server to be shut down first.
